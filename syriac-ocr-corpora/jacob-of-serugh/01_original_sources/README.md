# Original source volumes

This directory records the public source volumes from which the Jacob of
Serugh page images were prepared. The uncropped page images are not duplicated
in this repository because the original volumes are available online; the
repository instead preserves the prepared OCR input crops and the resulting
Qoruyo OCR output.
