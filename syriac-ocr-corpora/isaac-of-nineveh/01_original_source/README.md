# Original Source

The uncropped source edition documented by this corpus is:

[Isaac of Nineveh, *De perfectione religiosa*](https://archive.org/details/deperfectionerel0000isaa)

The Archive.org record is the external source record for the edition. The
uncropped page images are included in `uncropped_photos/` for reproducible
inspection and comparison with the OCR.

The files preserve the source page framing and use filenames keyed to the
source page number. The corresponding page-level OCR is in
[`../03_qoruyo_ocr/`](../03_qoruyo_ocr/), and the
[merged transcription](../isaac-of-nineveh-complete-qoruyo-ocr.txt) is at the
corpus root.
