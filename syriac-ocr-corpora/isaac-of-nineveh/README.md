# Isaac of Nineveh: Source Images

This directory currently contains the uncropped page images for the Isaac of
Nineveh edition documented at
[Archive.org](https://archive.org/details/deperfectionerel0000isaa).

## Contents

- `01_original_source/README.md`: documentation for the external source
  edition.
- `01_original_source/uncropped_photos/`: 638 page images preserving the
  original page framing and page numbering.

Prepared crops and Qoruyo OCR are intentionally not included in this release.
They can be added later as separate processing stages without changing the
source-image record.

For a general reproducible route from uncropped page images to prepared crops,
see the repository's optional
[`document-ai-preprocessing/`](../../document-ai-preprocessing/) workflow. It
can apply the same layout-based preparation approach to this or another
edition, followed by human review before OCR.
